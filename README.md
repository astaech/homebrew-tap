# Astaech Tap

## How do I install these formulae?

`brew install astaech/tap/<formula>`

Or `brew tap astaech/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "astaech/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
